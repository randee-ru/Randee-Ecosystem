## Технический стек
| Компонент | Решение |
|---|---|
| Canvas рендеринг | CSS transforms + absolute positioning |
| Хранилище | SQLite (Prisma) — JSON поле data |
| Изображения | base64 в JSON (фаза 2) → S3/file upload (фаза 3) |
| Undo/Redo | Stack из состояний (max 50) |
| Автосохранение | debounce 1.5s → PUT /api/design/[fileId] |
| Экспорт в PNG | html2canvas / dom-to-image (фаза 6) |
| Экспорт в SVG | Генерация из дерева слоёв (фаза 6) |
