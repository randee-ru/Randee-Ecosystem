# Cloud Overview

Randee Cloud предоставляет:

- preview deploys;
- collaboration;
- project backup/sync;
- telemetry и audit trail.

Cloud слой не должен быть обязательным для on-prem использования Randee в Bitrix-проектах.

MVP реализован через `@randee/cloud` и покрывает:

- cloud projects и members;
- preview URL provisioning;
- project sync state;
- audit trail.
