# Урок 03: Экспорт в Bitrix

## Цель

Экспортировать страницу в структуру Bitrix-компонентов.

## Шаги

```bash
npm run randee:export:sample
```

## Что проверить

- `dist/bitrix-site/local/components/randee/hero`
- `dist/bitrix-site/local/components/randee/faq`
- `dist/bitrix-site/local/components/randee/catalog.section`
- `dist/bitrix-site/randee-export-manifest.json`
