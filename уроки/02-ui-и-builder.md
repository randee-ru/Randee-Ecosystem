# Урок 02: UI и Builder

## Цель

Запустить UI и открыть Builder MVP.

## Шаги

```bash
npm run dev
```

Откройте:

- `/` — стартовая страница
- `/builder` — Builder MVP

## Практика

1. Добавьте блок `Hero`.
2. Добавьте блок `FAQ`.
3. Продублируйте `Hero`.
4. Перетащите блоки в другой порядок.
5. Измените props в Inspector.
